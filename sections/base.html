LLM Base

A Base model is the large pretrained transformer (or variant) trained on broad corpora. It provides the general-purpose language understanding and representations used as the starting point for fine-tuning and downstream tasks.

Key points

  • Model architecture (decoder-only, encoder-decoder, etc.)
  • Tokenizer format and vocabulary
  • Model weights file formats (.bin, .safetensors)
  • Model size trade-offs (parameters vs. latency)
