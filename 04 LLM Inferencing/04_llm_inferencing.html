Understanding Inference

Inference is the process of using a trained model to generate outputs (predictions, text, tokens) for user prompts. It is the runtime bridge between training and delivering responses.

Key topics: latency, cost, scalability, and trade-offs between model size and serving efficiency. Quantization, batching, and optimized runtimes are commonly used to reduce cost and latency.
